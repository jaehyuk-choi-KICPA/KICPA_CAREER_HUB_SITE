- **빅4 신입 공채 ⭐ 특집** — '방금 올라온 공고' 옆 책갈피를 누르면, 삼일·삼정·안진·한영 **빅4 신입 회계사 정기채용 일정**(접수기간·D-day·지원 링크)을 한곳에 모아 봅니다. 감사/비감사로 일정이 나뉘면 두 줄로, 일정이 미정이면 '업로드 예정'으로 표시합니다. **배경:** 정기채용은 시즌이 정해져 있어 흩어진 일정을 한눈에 모아 두면 준비에 유용합니다. (각 법인 공식 페이지 기준으로 **수동 큐레이션**하며 시즌마다 갱신.)

## v1.1 — 2026-06-21
> 채용 필터를 '직무'에서 **자격요건·채용구분**으로 바꾸고, 채용 카드와 상단 헤더·검색 영역을 잡코리아풍으로 깔끔하게 다시 짰습니다.

### 🎨 UI 개선
- **채용 카드 전면 개편** — 좌상단에 **법인 + 채용구분 + 자격구분** 태그, 제목은 검정으로, 그 아래 한 줄에 **`게시일 | 기관명 | D-day`**를 담았습니다(좁은 모바일 2열에서도 한 줄 유지 — 기관명만 말줄임). **D-day는 진한 빨강**으로 통일하고 박스를 없앴으며, **당일마감만 'D-0' 박스**로 강조합니다. **배경:** 정보가 흩어져 산만하던 카드를 제목 중심으로 정리하고, 한눈에 마감을 읽도록 했습니다. (게시일이 없는 공고는 처음 수집된 날짜로 채웁니다.)
- **상단 헤더 재배치** — 1행은 **로고 … 안내·채용알림**, 2행은 가운데 **채용공고·기사·인사이트 + 다크모드**와 우측 **'최근 서치'**로 나눴습니다. '최근 업데이트' 표기를 **'최근 서치'**로 바꿨습니다.
- **검색·필터·정렬을 옅은 파란 묶음으로** — 채용 화면의 검색창·필터·정렬을 하나의 파란 톤 박스로 묶어 잡코리아처럼 한 덩어리로 보이게 했습니다.
- **카드 전체 클릭** — 이제 카드 아무 곳이나 누르면 살짝 눌리는 효과와 함께 공고/기사로 이동합니다(기사 '동일 주제 펼치기'는 그대로 펼침).
- **기사 제목도 검정으로**, 기사 탭 상단의 안내 문구는 덜어냈습니다.
- **'방금 올라온 공고'(24시간)** — '새로 올라온 공고' 칸 기준을 **최근 24시간**으로 좁히고 이름·설명을 다듬었습니다.
- **안내 탭 보강** — 페이지 소개를 상단 강조 박스로 올리고 설명을 정리했습니다.

### ✨ 새로운 기능
- **자격요건·채용구분 2축 필터** — 채용 필터에서 '직무(딜/감사/택스/기타)'를 빼고, **자격요건(수습CPA / 자격무관)**과 **채용구분(인턴 / 정규직 / 계약직 / 파트타임)** 두 축을 새로 넣었습니다. **배경:** 타깃이 수습공인회계사라, "이 공고가 합격자(수습CPA)를 대상으로 하는가"와 "어떤 형태로 뽑는가"가 직무 구분보다 더 결정적이라고 보았습니다.
- **채용알림(웹 푸시)** — 채용 화면 검색줄의 **🔔 채용알림**에서 새 공고 푸시 알림을 신청합니다. **전체(일반·인턴 포함) / 수습CPA 전용** 중 받을 범위를 고를 수 있어요. **배경:** 매번 들어와 새로고침하지 않아도, 관심 범위의 새 공고가 올라오면 브라우저 알림으로 바로 받도록 했습니다. (구독은 이 기기에 저장, 브라우저 알림 설정에서 해제 가능.)

## v1.09 — 2026-06-18
> 빅펌 인사이트를 법인별 박스로 새로 짜고, 탭·필터·법인 표기를 더 깔끔한 톤으로 통일했습니다.

### 🎨 UI 개선
- **빅펌 인사이트 개편 — 법인 4박스** — 평평하던 인사이트 목록을 **삼일PwC·삼정KPMG·Deloitte안진·EY한영 4개 박스**로 바꿨습니다(PC는 2×2, 모바일은 한 줄씩). 박스마다 법인색을 연하게 두른 모던한 테두리가 들어가고, 법인 구분 필터탭은 없앴습니다(자리엔 '오늘의 인사이트' 안내). **배경:** 여러 법인 글이 뒤섞여 산만하고 '금일' 표시가 부정확해, 법인별로 깔끔히 나눠 보이게 했습니다.
- **상단 탭·기사 구분탭을 '밑줄 탭'으로** — 채워진 알약 버튼 대신, 글자 + 선택 시 밑줄로 표시합니다(탭끼리 여백). 기사 카테고리 탭도 동일.
- **법인명을 색글씨 풀네임으로 통일** — 채용 카드의 법인 표기를 채운 배지 대신 **법인색 글씨(삼일PwC·Deloitte안진 등 풀네임)**로 바꿨습니다(필터 패널은 그대로). '새로 올라온 공고'도 풀네임으로(글자색은 기본, 법인색 점으로 구분).
- **미선택 탭에 옅은 밑줄** — 상단 탭·기사 구분탭의 선택되지 않은 항목에도 옅은 회색 밑줄을 칸마다 간격을 두고 그어, 탭 전환 시 어디가 탭인지 더 잘 보이게 했습니다.
- **채용 카드 법인명 영문화(모바일 공간 절약)** — 채용 카드의 법인 표기를 영문만 남겼습니다(삼일→PwC·삼정→KPMG·안진→Deloitte·한영→EY, 로컬/기타는 한글). **배경:** 모바일 2열에서 풀네임이 길어 카드가 비좁던 문제를 풀었습니다. ('새로 올라온 공고'는 풀네임 유지.)
- **'금일 기사' 미니박스 절반 크기** — 우측 단독으로 남아 휑하던 '금일 기사' 박스를 절반 폭으로 줄였습니다.
- **'새로 올라온 공고' 흐림 기준 변경** — 이틀 전(어제) 공고를 흐리게 하던 날짜 기준을 없애고(어차피 최신순 정렬이라 충분), 대신 **내가 클릭해 본 공고만** 흐리게 표시합니다(어디까지 봤는지 한눈에, 이 브라우저에만 기억).

### ✨ 새로운 기능
- **인사이트 박스별 '오늘의 추천' + 펼치기** — 각 법인 박스가 **새로고침마다 무작위로 한 편**을 추천하고, **'펼치기(최신순)'**를 누르면 그 법인 글 전체가 좌르륵 펼쳐집니다.
- **'새로 올라온 공고' 진짜 올라온 순 정렬** — 게시일(날짜)만으로는 같은 날 공고 순서가 뒤섞였는데, 이제 **실제로 올라온(발견된) 시각 기준**으로 정렬해 방금 올라온 공고가 위로 옵니다.
