# 회법몬 패치노트

빅4·로컬 회계법인 채용공고와 회계·세무·딜 업계 기사를 한곳에 모아주는 **회법몬**의 빌드별 변경 기록입니다.
좌상단에 표시되는 **공식 버전은 UI 개선·새 기능이 추가될 때** 올라갑니다.
(수집/스크랩 엔진의 내부 개선은 버전과 별개로 [SCRAPER_LOG.md](SCRAPER_LOG.md)에서 관리합니다.)

> 작성 규칙: 최신 버전을 맨 위에. 각 버전은 **🎨 UI 개선 / ✨ 새로운 기능** 두 분류를 항상 표기(없으면 '변경 없음').
> 각 항목은 "무엇이 바뀌었는지 + 왜(배경)"를 함께 적습니다.

## v1.08 — 2026-06-18
> '새로 올라온 공고'에서 가장 최근 글을 한눈에 알아보고, 기사 탭에 새 글 알림 점을 더했습니다.

### 🎨 UI 개선
- **'새로 올라온 공고' 맨 위 '최신' 배지** — 패널은 최신순으로 정렬되는데, 가장 위 공고에 작은 '최신' 배지를 달아 맨 위가 가장 최근임을 한눈에 알 수 있게 했습니다. **배경:** 어제 공고를 흐리게 처리한 것에 더해, 정렬 기준을 직관적으로 드러내기 위해서입니다.
- **기사 '새 글' 빨간 점** — 새로 올라온 기사 카드에 빨간 점이 뜨고, **그 기사를 보면(제목 클릭, 또는 '동일 주제 기사' 펼치기) 점이 사라집니다.** 안 본 새 기사가 남아 있으면 상단 **'기사' 탭에도 빨간 점**이 떠서, 탭을 열어보면(또는 새 기사를 하나 눌러보면) 사라집니다. 본 기록은 **이 브라우저에만** 저장됩니다(로그인 없는 사이트라 기기별로 관리). **배경:** 새 소식이 올라왔는지 탭만 봐도 알 수 있게, 그리고 이미 본 건 표시가 사라지게 했습니다. (인사이트 탭 알림은 다음 버전에서 정리 예정.)

## v1.07 — 2026-06-18
> 같은 사건을 다룬 중복 기사를 하나로 묶어, 목록이 도배되지 않고 깔끔하게 정리됩니다.

### 🎨 UI 개선
- **'동일 주제 기사 N개' 펼침 카드** — 중복으로 묶인 기사는 카드 우측 하단의 알약 토글로 접혀 있다가, 누르면 펼쳐지며 나머지 기사들의 제목·출처가 뜨고 각각 원문으로 바로 이동할 수 있습니다. **배경:** 도배되던 목록을 깔끔하게 정리하면서도, 어느 매체가 다뤘는지는 한눈에 펼쳐 확인할 수 있게 했습니다.

### ✨ 새로운 기능
- **중복 기사 묶기(군집화)** — 같은 사건을 여러 매체가 보도해 제목이 겹치면 가장 최근 기사를 대표로 묶고 나머지를 통합합니다. 특히 딜·M&A·세무 쪽에서 같은 뉴스가 매체만 바꿔 여러 번 뜨던 도배가 정리되고, 어느 매체가 다뤘는지 정보는 모두 보존됩니다. (묶인 중복 기사는 '금일 기사 수'에 중복으로 세지 않습니다.)

## v1.06 — 2026-06-18
> '금일' 공고 패널을 최근 2일 창으로 넓히고, 어제 공고는 흐리게 보여 한눈에 신선도를 구분하게 했습니다.

### 🎨 UI 개선
- **'금일 올라온 공고' → '새로 올라온 공고'(최근 2일)** — 우측 패널이 오늘 하루만이 아니라 **어제·오늘 이틀치** 공고를 보여줍니다. 자정이 지나면 창이 하루씩 자동으로 이동합니다(예: 18일엔 17·18일, 19일엔 18·19일). **배경:** 하루치만 보여주니 자정 직후나 공고가 뜸한 날엔 패널이 자주 비어 허전했고, 막 지나간 어제 공고를 놓치기 쉬웠습니다.
- **어제 공고는 흐리게** — 패널 안에서 가장 최근 날짜(오늘)가 아닌 공고는 글씨를 연하게 표시합니다. **배경:** 같은 패널에 이틀치가 섞여도 어느 게 막 올라온 따끈한 공고인지 색만으로 직관적으로 구분되게 했습니다.

### ✨ 새로운 기능

## v1.05 — 2026-06-18
> 모바일 화면과 필터를 전면적으로 다듬고, 화면 정보 밀도를 정리했습니다.

### 🎨 UI 개선
- **모바일 레이아웃 재구성** — 검색·필터·정렬 바를 '금일' 박스 아래로 내리고, 채용공고는 한 줄에 2장·기사/인사이트는 1장으로 정리했습니다. **배경:** 모바일에서 공고가 한 줄에 한 장씩만 떠 화면이 너무 좁고 한눈에 안 들어왔습니다.
- **카드 표시 정리** — 마감(D-day)은 카드 오른쪽 위로, 'NEW' 글자 배지는 없애고 신규는 카드 왼쪽 초록 테두리로만 표시합니다. 당일 마감은 모바일에서 '오늘/마감' 두 줄로 표시합니다. **배경:** 정보량을 줄이고, 글자가 길어 카드 칸이 무너지는 문제를 막았습니다.
- **긴 제목 줄임** — 공고·기사 제목이 길면 최대 3줄까지만 보이고 말줄임 처리합니다. **배경:** 카드 높이가 제각각이던 문제를 정리했습니다.
- **진행 상태에 '상설' 추가** — 진행중 / 마감 / 상설(상시채용)로 구분합니다. **배경:** 마감일이 없는 상시 공고를 따로 보고 싶다는 필요를 반영했습니다.
- **법인 필터 숫자가 상태에 따라 바뀜** — 진행중·마감·상설을 누르면 그 상태에 해당하는 법인별 건수만 보여줍니다. 직무 필터의 괄호 숫자는 없앴습니다. **배경:** 지금 보는 조건의 실제 분포를 바로 알 수 있게 하고, 불필요한 숫자는 정리했습니다.
- **활성 필터 요약칩 + 결과 건수** — 선택한 필터를 결과 위에 칩으로 보여주고 ✕로 바로 해제할 수 있으며, 현재 결과 건수를 즉시 표시합니다. **배경:** 필터 패널을 다시 열지 않고도 무엇이 걸려 있는지 보고 끄게 했습니다.
- **빅펌 인사이트 태그 개선** — 모든 카드에 똑같이 달리던 '인사이트' 태그를 없애고, 발행 법인(삼일·삼정·안진·한영)별 색상 태그로 바꿨습니다. **배경:** 어느 법인 자료인지 한눈에 구분되게 했습니다.
- **로고·버전 글자 확대**와 **'최근 업데이트' 표시를 알약 모양 칩**으로 다듬었습니다. **배경:** 첫인상과 가독성을 높였습니다.

### ✨ 새로운 기능
- **로고 클릭 시 새로고침** — 좌상단 '회법몬'을 누르면 채용공고 탭 맨 위로 이동하면서 최신 데이터를 다시 불러옵니다. **배경:** 화면을 오래 켜둔 상태에서도 손쉽게 갱신할 수 있게 했습니다.
